Instrucciones:

  1. Crear un servicio que se encargue de hacer peticiones GET
  2. Crear componente (customSelect) al que se le pasará información
  3. Hacer petición GET a través de servicio creado: ./assets/window.json
  4. Usar atributo style de elemento con type = select de array controls como estilo de un select dentro del componente creado (customSelect)
  5. Hacer petición GET a través de servicio creado: ./assets/formats.json
  6. Llenar select de customSelect con array de formatos, mostrando sólo aquellas cuyo atributo "type" sea igual a 1
  7. Crear una forma dentro de customSelect (de preferencia usar reactive forms) que se llene con los campos del objeto seleccionado [id, format, formatDisplay, type]
  8. Permitir modificar solamente campo formatDisplay (los demás campos deben permanecer inhabilitados)
  9. Crear botón que permanezca inhabilitado hasta que se modifique el valor de formatDisplay y que capture los valores de la forma una vez sea habilitado
  10. Mostrar valores de la forma
